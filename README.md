# UI Inspector

A browser console tool for QA annotation on React apps.
Click any element to identify its React component, log the bug, and export reports.

## Commands
- `window.inspectUI()` — Start/stop inspect mode
- `window.viewIssues()` — View all logged issues
- `window.copyIssues()` — Copy issues to clipboard
- `window.clearIssues()` — Clear all issues
